# Temimi 视频平台项目功能分析

## 项目概述
Temimi 是一个类似 B站 的视频分享平台，包含三个子项目：
- **temimi-backend**: Spring Boot 后端服务
- **temimi-admin**: Vue3 管理后台
- **temimi-client**: Vue3 用户端

---

## 三、后端缺失的API

根据前端页面需求，后端可能需要补充以下API：

### 内容管理API
```
POST   /api/admin/carousel          - 添加轮播图
GET    /api/admin/carousel/list     - 获取轮播图列表
PUT    /api/admin/carousel/{id}     - 更新轮播图
DELETE /api/admin/carousel/{id}     - 删除轮播图

POST   /api/admin/hot-search        - 添加热搜
GET    /api/admin/hot-search/list   - 获取热搜列表
PUT    /api/admin/hot-search/{id}   - 更新热搜
DELETE /api/admin/hot-search/{id}   - 删除热搜

GET    /api/admin/tag/list          - 获取标签列表
POST   /api/admin/tag               - 添加标签
PUT    /api/admin/tag/{id}          - 更新标签
DELETE /api/admin/tag/{id}          - 删除标签
```

### 文章模块API
```
POST   /api/article/upload          - 上传文章
GET    /api/article/{id}            - 获取文章详情
GET    /api/article/user/{uid}      - 获取用户文章列表
DELETE /api/article/{id}            - 删除文章
GET    /api/admin/article/list      - 管理后台文章列表
POST   /api/admin/article/review    - 文章审核
```

### 审核模块API
```
GET    /api/admin/avatar/list       - 待审核头像列表
POST   /api/admin/avatar/review     - 头像审核

GET    /api/admin/dynamic/list      - 待审核动态列表
POST   /api/admin/dynamic/review    - 动态审核
```

### 申诉/举报API
```
POST   /api/appeal/submit           - 提交申诉
GET    /api/appeal/my               - 我的申诉列表
GET    /api/admin/appeal/list       - 申诉列表(管理)
POST   /api/admin/appeal/handle     - 处理申诉

POST   /api/report/submit           - 提交举报
GET    /api/admin/report/list       - 举报列表(管理)
POST   /api/admin/report/handle     - 处理举报
```

### 数据统计API
```
GET    /api/admin/stats/overview    - 平台数据概览
GET    /api/admin/stats/trend       - 数据趋势
GET    /api/creator/stats           - 创作者数据统计
```

## 四、数据库表结构补充建议

根据未实现功能，可能需要新增以下数据表：

```sql
-- 轮播图表
CREATE TABLE carousel (
    id INT PRIMARY KEY AUTO_INCREMENT,
    image_url VARCHAR(500) NOT NULL,
    link_url VARCHAR(500),
    title VARCHAR(100),
    sort_order INT DEFAULT 0,
    status TINYINT DEFAULT 1,
    create_time DATETIME
);

-- 热搜表
CREATE TABLE hot_search (
    id INT PRIMARY KEY AUTO_INCREMENT,
    keyword VARCHAR(50) NOT NULL,
    heat INT DEFAULT 0,
    type TINYINT DEFAULT 1,
    create_time DATETIME
);

-- 文章表
CREATE TABLE article (
    id INT PRIMARY KEY AUTO_INCREMENT,
    uid INT NOT NULL,
    title VARCHAR(100) NOT NULL,
    content TEXT,
    cover VARCHAR(500),
    status TINYINT DEFAULT 0,
    view_count INT DEFAULT 0,
    create_time DATETIME
);

-- 申诉表
CREATE TABLE appeal (
    id INT PRIMARY KEY AUTO_INCREMENT,
    uid INT NOT NULL,
    type TINYINT NOT NULL,
    target_id INT NOT NULL,
    reason TEXT,
    status TINYINT DEFAULT 0,
    create_time DATETIME,
    handle_time DATETIME
);

-- 举报表
CREATE TABLE report (
    id INT PRIMARY KEY AUTO_INCREMENT,
    uid INT NOT NULL,
    type TINYINT NOT NULL,
    target_id INT NOT NULL,
    reason TEXT,
    status TINYINT DEFAULT 0,
    create_time DATETIME,
    handle_time DATETIME
);
```
